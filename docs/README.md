## Diagram Formats

The documentation includes diagrams in multiple formats:

- **Mermaid:** Interactive diagrams (renders in GitHub, VS Code, etc.)
- **PlantUML:** UML diagrams (requires PlantUML viewer)
- **ASCII Art:** Text-based diagrams (universal compatibility)

## Viewing Diagrams

### Mermaid Diagrams
- **GitHub:** Automatically rendered
- **VS Code:** Install "Markdown Preview Mermaid Support" extension
- **Online:** Use [Mermaid Live Editor](https://mermaid.live/)

### PlantUML Diagrams
- **VS Code:** Install "PlantUML" extension
- **Online:** Use [PlantUML Server](http://www.plantuml.com/plantuml/uml/)
- **Local:** Install PlantUML and use with your preferred editor
